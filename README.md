# Mini Crawler Pre-Work
### Windows
```console
mvnw clean package

java -Dfile.encoding=UTF-8 -jar target\web-crawler-1.0-SNAPSHOT-jar-with-dependencies.jar
```
### Linux/MacOS
```console
./mvnw clean package

java -Dfile.encoding=UTF-8 -jar target/web-crawler-1.0-SNAPSHOT-jar-with-dependencies.jar
```

See result in folder output/data.csv
